上海交大多措并举践行“学在交大”的育人理念[图]

2021年09月26日 责任编辑：刘婧怡 

人民日报

人民日报报道 为更好地推进教学改革，全方位服务师生学习，上海交通大学通过线上课程建设、“名师教你学”系列讲座、教学楼升级改造等方面开展多种工作，为教学保驾护航。通过线上线下混合式教学改革，深入贯彻“金课”理念，打造一批优质课程。邀请名师为新生学习答疑解惑，助力交大学子顺利完成新学期学业。对教学楼进行升级改造，为师生提供舒适便捷的学习环境。上海交大多措并举践行“学在交大”的育人理念。

线上线下混合式“教改”，55门课程获评“金课”

结合双一流建设，上海交大建立具有交大特色的一流课程体系，按照“自上而下”和“自下而上”相结合，校级、市级、国家级逐步进阶的方式，有计划、有重点、分步开展建设，形成一流课程“蓄水池”。学校整体布局、院系分类建设，以在线课程建设为主线，在学校支持的基础上开展专业核心课的在线课程建设，进行线上线下混合式教学改革。

学校“大教务”运行机制协同推进资源整合，制定《精品在线课程建设管理办法》、金课建设规划，运用中国大学MOOC、“好大学在线”和CANVAS@SJTU平台，推进混合式课堂教学改革。教务处深入院系开展“金课”理念和课程建设宣讲，教学发展中心推出一流课程建设量规，引导一线教师开展教学改革，提升课程质量，教育技术中心负责一流课程建设中所需的课程资源、平台建设服务。

上海交通大学于2013年在国内率先开展慕课建设及推广，截止目前，共上线在线课程（慕课）165门。教育部于2017年教育部开始认定国家级精品在线开放课程（即线上金课），2019年开始认定国家级一流课程（五类金课）。截止目前，上海交大共有55门课程被认定为国家级一流课程（金课）。

“名师教你学”系列讲座，为新生学习保驾护航

“名师教你学”系列讲座由上海交通大学教学发展中心主办，面向大一新生，邀请各学科名师讲解大学里数学、化学、英语、物理等课程的学习。旨在帮助新生明确学习目标、掌握学习方法、养成学习习惯、提高学习效率，通过名师导航，助力新生学业顺利成长。

“名师教你学”第一讲由2021年“交大名师”、数学科学学院特聘教授、自然科学研究院常务副院长、致远荣誉计划数学方向项目主任王维克开讲，主题为《如何在大学里学好数学课》。微积分等大学数学的课程内容对于学生数学思维习惯的养成、具备现代文化素养非常重要。然而，数学课程中会有一些抽象概念成为学生深入理解的“拦路虎”。王维克教授通过分析经典的数学案例，以通俗易通的语言和生动的现场互动，循序渐进地阐明了数学中的抽象概念。不仅帮助新生对高等数学形成准确的认知，还对如何学好大学数学、形成数学抽象思维习惯进行了有效的启发引导，极大激发了新生对数学学习的兴趣和探索。


教学楼升级改造：让教室成为学校最美的风景

为坚持立德树人根本任务、持续深化教育教学改革，上海交通大学先后对东中院、东下院教学楼进行升级改造，通过空间重塑，营造出积极向上、开放灵活的空间体系，更好地实现课内课外、校内校外、国内国外、线上线下的融合，适应新时代下“教”和“学”的方式。

改造后的东中院、东下院“颜值在线”：休读空间、屋顶花园、露天大平台成为学生们的“心头好”。在东中院改造之前，教务处、教育技术中心、基建处也广泛征询了师生意见，在教室课桌下增添插座、教师休息室设置储物柜、卫生间配备洗手液等，给教学楼增添了“温度”。

技术赋能，创新教育，让学习成为一种新的体验。教学楼大厅信息发布系统、智能储物柜为师生提供便利。教室内的智能控制系统可根据课程表开启多媒体设备，空调、灯光、新风设备等，根据每日天气情况进行智能调控，为师生的学习营造舒适的外部环境。东下院的升级改造对教室进行降噪处理、配备LED大屏和互联黑板、将部分固定桌椅改成活动桌椅、小组多屏互动教室的设置为提高教学效率，方便师生协作交流。

教育技术中心着力教学楼升级改造的同时，也注重在线教学平台建设。Canvas@SJTU的上线、课堂教学在线直播系统的设置，为疫情背景下中外学生的课程学习提供便利。2021年，上海交通大学启动“交·通全球课堂”计划，学校改建线上线下融合式教室，让学生不出国门也有机会修读“全球课程”。

除了硬件改造、软件升级之外，东中院、东下院在楼宇文化建设上也做了精心的规划。东中院内庭，《交大映像》，东下院西入口院士墙、东入口“校长奖”文化墙，东中院阳光厅教书育人奖获得者风采展示墙，充分展现出交大独特的教育理念和育人目标。
