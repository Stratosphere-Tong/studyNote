2. 公交：958路；

3. 校园巴士早晚有班次往返于东川路地铁站与菁菁堂广场之间（7:24/7:39/7:54/8:09东川路地铁站发车；16:55/17:00/17:25/17:40/18:00/18:15/18:40/19:25/20:25/20:55菁菁堂发车，请提前候车）；

4. 江川7路约30分钟发车一次，早高峰7:10-8:55，晚高峰16:25-18:10加密营运班次。
