  • 上海交通大学“海铃计划”取得重要进展

    近日，“海铃计划”探路者项目团队完成各项预定海试任务并安全抵沪。本航次由上海交通大学李政道学者徐东莲担任首席科学家，海洋工程学者田新亮担任领队，共有来自上海交通大学、北京大学、清华大学、中国科学技术大学、自然资源部第二海洋研究所等机构的三十余位科研人员与技术人员共同参与。本次科考为“海铃计划”的后续推进奠定了坚实的基础。

  • 市委书记李强调研中欧国际工商学院

    市委书记李强9月16日下午在调研中欧国际工商学院、上海纽约大学时指出，当前，上海正在深入贯彻落实习近平总书记在浦东开发开放30周年庆祝大会上重要讲话精神，加快建设具有世界影响力的社会主义现代化国际大都市，希望中欧国际工商学院、上海纽约大学始终坚持正确办学方向，更好发挥开放办学优势，努力走在中外合作办学前列，为城市发展导入更多国际资源、集聚更多优秀人才、孕育更多创新成果、提供更多智力支持，在浦江之畔绽放出新的精彩。

  • “谦纯优秀青年学者”基金景昱医疗捐赠签约...

    9月16日上午，上海交通大学“谦纯优秀青年学者”基金景昱医疗捐赠签约仪式在上海交通大学闵行校区转化医学大楼举行。“谦纯优秀青年学者”基金捐赠人、英国皇家工程院院士、上海交大医疗机器人研究院院长、生物医学工程学院院长杨广中教授，苏州景昱医疗器械有限公司董事长宁益华学长出席签约仪式，迈瑞医疗集团联合创始人、校董成明和学长，软银中国管理合伙人华平学长，精裕投资董事长施永敏学长，上海医学创新发展基金会秘书长王波女士应邀出席。校党委书记杨振斌，副校长徐学敏，相关部处、学院及研究中心负责人参加仪式并见证捐赠签约。活动由徐学敏主持。

  • 上海交通大学-亚太空间合作组织APSCO SSS-2...

    9月24日上午，上海交通大学-亚太空间合作组织APSCO SSS-2A卫星出征仪式在交大空天大楼举行。上海交通大学校长林忠钦院士、副校长朱新远，交大先进技术与装备研究院院长曾小勤、副院长刘冬，以及航空航天学院领导、智能卫星团队师生参加出征仪式。合作单位上海宇航系统工程研究所所长刘鲁江、长二丁项目办副主任金煜受邀出席活动。

  • “品物游心：中国文人的生活与艺术”展在上...

    俯仰天地间，乘物以游心。9月24日，由上海交通大学和上海博物馆主办、上海交通大学博物馆承办的“品物游心——中国文人的生活与艺术”展在上海交通大学闵行校区文博楼拉开帷幕。上海博物馆馆长杨志刚、上海大学党委副书记段勇等校外来宾，上海交通大学校长林忠钦、原党委书记王宗光、原校长谢绳武、原党委书记马德秀、党委常委、副校长张安胜等学校领导，以及上海博物馆展览团队、相关博物馆及兄弟院校领导，上海交通大学博物馆馆长陈克伦、钱学森图书馆馆长钱永刚等学校有关院系部处和直属单位的领导及师生代表100余人出席了开幕式。
