上海交通大学2021年第三季度学生思想政治工作大会举行

2021年09月26日 责任编辑：汤翔鹰 

9月23日，上海交通大学2021年第三季度学生思想政治工作大会在闵行校区李政道图书馆报告厅举行。校党委副书记、副校长王伟明，学指委秘书长、学生处处长、学生工作党委书记侯士兵，学指委处班子，院（系）党委（总支）副书记，全体思政教师及辅导员参会。大会由学生处副处长龚强主持。

王伟明肯定了学生工作队伍前一阶段在军训育人、国防教育、实践育人、征兵工作以及疫情防控等方面的工作，向全体思政教师、辅导员的辛勤付出表示敬意和感谢。针对新形势下的学生工作，他从如何与时俱进、如何做到本领过硬、如何把工作落实落细三个方面提出要求。他强调，思政教师要提高政治站位，落实好巡视巡察整改工作，从为国家培养人才的高度出发，与时俱进，牢记立德树人的使命；思政教师要练就过硬本领，明确人才培养的定位，不拘泥于惯性的工作方式，注重方式方法的迭代，学会挖掘更多的资源，注重高质量学习成果转化；思政教师要在工作的落实落细上肯花功夫、肯下辛苦，关心到每个工作对象，建立更加完善的工作体系，从上率下、以身作则，助力每位思政教师和学生成长。面对暑期工作和新学期工作的无缝衔接，他鼓励在座的各位老师积极调整，合理分配时间，始终以高水平、高热情的状态投入到立德树人的工作中去。

 

侯士兵围绕加强国防教育、强化实践育人、推进新生教育、坚守疫情防控阵地四个方面总结了2021年暑期及开学季相关工作，并从落实巡视整改工作、学生价值引领、队伍建设、学在交大、管理服务、安全稳定等方面部署了新学期重点工作。他希望各位思政教师及辅导员能够再接再厉，聚焦学生工作主责主业，扎实践行育人神圣使命，做思想政治工作的主攻手、学生管理的主导者、学生成长的主心骨。

 

龚强宣读《关于嘉奖2021年学生军训及迎新工作集体和个人的决定》。大会向获得嘉奖的学生军训及迎新工作集体和个人颁奖。

凯原法学院谢焕作为军训工作代表发言。她结合今年两次参训经历，分享了自己对思政教师责任和使命的思考。材料科学与工程学院党委副书记沈小丹作迎新工作专题分享，她从如何高效准确完成台账、如何做好迎新工作以及如何做好新生入学教育等方面分享了学院的经验与思考。她表示，万事开头难，但在全体思政教师的全情投入下，相信每个困难都能够迎刃而解。
