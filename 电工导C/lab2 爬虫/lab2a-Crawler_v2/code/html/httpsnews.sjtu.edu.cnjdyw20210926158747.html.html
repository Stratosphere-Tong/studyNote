交大要闻

武警上海总队执勤第二支队领导访问上海交大

2021年09月26日 责任编辑：康健 

9月22日下午，武警上海总队执勤第二支队政委吴方泽带领支队班子一行访问上海交大。校党委书记杨振斌，副书记、副校长王伟明会见了吴方泽一行，双方围绕学生国防教育、双拥工作等内容进行了座谈交流。

吴方泽对上海交大跨越三个世纪的悠久历史和为国家培养了大批人才表示敬佩。他强调，二支队将一如既往地在培养学生国防和国家安全意识方面提供支持，用好支队官兵的战斗故事，培养学生家国情怀。他表示，希望交大能经常派出专家、学者到二支队授课，不断提高部队官兵的理论水平和思想素质。

杨振斌对二支队在今年本科生军训中给予的大力支持表示由衷感谢。他强调，承训官兵的优异表现给师生们留下了深刻印象，和学校一起给学生上了一堂军训育人课。杨振斌表示，学校将不断推进与二支队的合作，支持部队建设，同时持续加强学生的国防和国家安全意识，培养学生家国情怀。

座谈会前，吴方泽一行在王伟明及学指委、武装部、档案文博管理中心等部门负责同志陪同下，参观了学校校史馆。

赵建世
人民武装部
