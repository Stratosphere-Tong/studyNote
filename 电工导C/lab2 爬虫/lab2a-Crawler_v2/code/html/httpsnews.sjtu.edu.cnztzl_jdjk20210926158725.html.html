交大金课

首批国家级线上一流课程

[交大金课]胡其图：大学物理_力学

— 交大金课
2021年09月26日 责任编辑：张文清 
2012年，全球慕课快速发展，上海交通大学于2013年在国内率先开展慕课建设及推广，截止目前，共上线在线课程（慕课）165门。教育部于2017年教育部开始认定国家级精品在线开放课程（即线上金课），2019年开始认定国家级一流课程（五类金课）。截至目前，全校共有55门课程被认定为国家级一流课程（金课）。

【金课名称】大学物理_力学

【任课教师】胡其图   顾志霞  李晟   邓晓   张小灵


【名师名片】

胡其图，上海交通大学物理与天文学院教授，国家级教学名师，国家级教学团队带头人，教育部高等学校大学物理课程教学指导委员会副主任委员，上海市物理学会大学生物理竞赛委员会主任委员。长期从事高等学校物理基础课程教学研究与课程建设，主要研究方向有物理学模拟与可视化、物理学增强现实技术、教育技术与信息集成等。获国家级教学成果二等奖3项(其中2项为第1获奖者)、国家级教学成果一等奖1项、国家级优秀教材二等奖2项（其中1项为第1作者）、国家级优秀教材一等奖1项、省部级教学成果奖8项、省部级科学技术进步奖二等奖1项。二次获全国优秀教师荣誉称号，一次获省部级先进工作者称号，获国务院政府特殊津贴，获宝钢优秀教师奖，获上海交通大学教学成果特等奖、优秀教材特等奖、优秀教师特等奖、教书育人奖、卓越教学奖、教学名师等多项奖。                                   

【课程介绍】

大学物理_力学在线课程是大学物理系列在线课程中的第一门课程，涉及的内容主要有质点运动学、牛顿运动定律、狭义相对论、动能定理和机械能守恒定律、动量定理和动量守恒定律、角动量定理和角动量守恒定律、刚体力学基础、振动力学基础、机械波等内容，本课程的表现形式主要有PPT动画叠加现场教学小窗口、全屏视频等，侧重于引导学生掌握基本的物理学概念、基本的物理学图像，探究物理学家建立物理学规律的基本方法和关键突破，提升教学内容的深度和广度，引导学生进行深入思考。

本课程引入计算机模拟可视化和交互性内容、引入物理演示实验内容，帮助学生从复杂的现象中、从抽象的概念中构建物理图像。以拓展研究性学习内容为突破点，唤起学生的好奇心与研究精神，产生学习与创造内驱力，营造一个有利于培养学生科学素养和创新意识的线上教学环境。

本课程由好大学在线平台和中国大学MOOC平台上线，累计有上海交通大学师生以及132所其他高校的3万余名师生先后注册该课程，在线上完成所需内容的相应学习。
