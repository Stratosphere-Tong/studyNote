交大要闻

上海交通大学-亚太空间合作组织APSCO SSS-2A卫星出征仪式举行

2021年09月25日 责任编辑：赵晶晶 

9月24日上午，上海交通大学-亚太空间合作组织APSCO SSS-2A卫星出征仪式在交大空天大楼举行。上海交通大学校长林忠钦院士、副校长朱新远，交大先进技术与装备研究院院长曾小勤、副院长刘冬，以及航空航天学院领导、智能卫星团队师生参加出征仪式。合作单位上海宇航系统工程研究所所长刘鲁江、长二丁项目办副主任金煜受邀出席活动。

领导和嘉宾在智能卫星技术中心聆听了师生对SSS-2A卫星研发过程的汇报，询问了卫星的软硬件设计、载荷实验等方面情况。林忠钦充分肯定了师生的成果，希望这颗卫星的成功研发能够激发更多学子的科技梦想。他指出，航空航天学院承载着学校更高的定位和更多的期待，学院要面向未来，结合交大实际，扎根中国大地，瞄准行业需求，展现交大智慧，贡献交大力量。刘鲁江表示，交大作为中国一流大学的代表，为国家航天事业做出了重要贡献，期望在未来能与交大继续扩大合作领域、拓展合作形式，推动建立更加深厚的合作关系。授旗仪式中，林忠钦向空天学院常务副院长、智能卫星技术中心负责人吴树范授旗，并与师生合影留念。

学院于2017年5月与亚太空间合作组织签订研发合同。SSS-2A卫星是交大第一颗由学生自主研发并主导载荷设计的卫星。搭载星载船舶自动识别系统（AIS）收集船舶信息和空间自适应组网模块开展空间窄带信息采集应用，搭载冷气微推模块、MEMS传感器、双折展开帆板，突破立方星能源限制，提升立方星在轨机动能力，拓展了立方星在轨应用场景。目前该卫星已通过一系列的卫星测试和可靠性验证工作，通过正样研发总结暨出厂评审，将于2021年10月中旬在太原发射中心搭载我国长征二号丁型火箭发射入轨。

APSCO SSS-2A卫星80%的单机为交大自主研发产品，智能卫星技术中心学生在导师的带领下，对设计的产品一次次迭代，精益求精，从本科生到博士生，每个学生都成功研发一块PCB板，都经历了系统设计、投产、测试、验证的研发全周期，充分挖掘了实践育人的价值。

立德树人是航空航天学院的立院之本，学院高度重视培养学生参与科技创新的意识与能力。多次组织学生参观珠海航展，到文昌、西昌等发射场开展实践，见证“北斗”、“天问一号”、“天和”核心舱的成功发射，联合航空航天单位开设“大国重器”课程，用“空天文化”做思政、育新人，让学生从前辈的奉献精神中找到人生的方向。这次，将有一颗自主研发的卫星发射上天，首次实现天地互联，相信对空天学子来说，这将成为人生记忆中最闪亮的一颗。
