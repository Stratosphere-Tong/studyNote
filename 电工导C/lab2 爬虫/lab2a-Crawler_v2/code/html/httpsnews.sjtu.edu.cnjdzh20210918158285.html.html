上海交大耿涌教授在顶级期刊《中国科学院院刊》组织专题：碳中和目标背景下的能源转型

2021年09月18日 责任编辑：王旭鑫 

气候变化问题的严峻性和紧迫性已成为世界各国的共识，“双碳”目标是我国基于推动构建人类命运共同体的责任担当和实现可持续发展的内在要求而作出的重大战略决策。以“双碳”战略为引擎，势必推动我国产业结构的变革和升级，从而实现工业、能源、技术、消费等方面的持续突破。为更好的促进绿色产业循环发展，优化社会经济体系，在《中国科学院院刊》编委，上海交通大学国际与公共事务学院副院长、环境科学与工程学院院长、上海交通大学-联合国工业发展组织绿色增长联合研究院院长耿涌教授指导推进下，在我国顶级期刊《中国科学院院刊》组织“碳中和目标背景下的能源转型”专题，邀请能源环境领域数位院士，从能源发展战略、能源变革、数字化技术、循环经济、能源转型的经济社会效益评估等方面探讨如何助推我国能源行业碳中和转型的达成。

中国工程院院士、上海交通大学-联合国工业发展组织绿色增长联合研究院理事长黄震，中国工程院院士、上海交通大学客座教授、上海交通大学-联合国工业发展组织绿色增长联合研究院首席教授陈晓红，耿涌教授，上海交通大学-联合国工业发展组织绿色增长联合研究院副教授魏文栋在该专题发表高水平论文。相关研究成果可为我国能源行业实现碳中和转型提供有益的政策启示，以确保能源结构更加清洁化和可更新，以及能源效率不断提高，为我国在高质量发展的同时实现碳中和目标提供有力支撑。

本次专题汇聚了专业领域重要的学术资源，为研究团队深入展开有关资源环境领域的系统研究搭建了重要框架。研究院将继续优化研究成果，形成具有创新可落地的技术能力，为我国“双碳”战略的有序推进，提供高水平决策平台。

上海交通大学—联合国工业发展组织绿色增长联合研究院自2020年11月揭牌成立以来，以绿色增长、循环经济、资源效率和环境管理及政策分析等为研究领域，注重学科交叉研究与学习交流借鉴，努力开展国际合作，打造国家高端智库。研究院积极整合交大资源优势，关注学术前沿和需求，力图实现高质量发展，推动技术的全球应用和学科发展。研究院以国际交流合作为特色，在联合国工发组织和上海交通大学的指导下，建立可持续技术应用中心，共同推动长三角地区乃至全国产业绿色增长转型及生态工业园建设，实现区域协同发展。
