[党史学习教育]“点亮初心，让青春与信仰同行”安泰经管学院学生党支部双线联动作好新生红色引领

2021年09月28日 责任编辑：孙瑞艳 

“当代中国青年是与新时代同向同行、共同前进的一代，生逢盛世，肩负重任。”百年征程波澜壮阔，百年初心历久弥坚，中国共产党立志于中华民族千秋伟业，百年恰是风华正茂。迎新启航，星火相传，正是青春选择奋进之时。2021秋季学期开学之际，安泰经济与管理学院学生党支部以“点亮初心，让青春与信仰同行”为主题，结合党史学习教育，线上线下双线联动，启动入党宣传引导，强化红色教育内容。

线上，在学院学生党建之家公众号上开通专栏，启动入党启蒙和理论学习教育，线上一站式了解入党流程，手把手指导入党志愿书书写规范。入学前在安泰学生党建研究会公众号上发布《“新”火汇聚｜点亮初心，让青春与信仰同行》系列推送，回顾党的百年奋进历程，帮助新生了解党的历史，了解入党全过程和入党申请书的书写，引导同学们树立理想抱负，勇担责任使命。通过理论学习专题推送，抓重点、要点，简明易懂地分享了党的十八大以来，以习近平同志为核心的党中央在领导推进新时代治国理政的实践中，提出的许多具有时代性、实践性、创造性的重大思想观点，营造良好学习氛围，为新生在政治上和思想上的进步指引航向。


线下，各学生党支部成立党员先锋队，学生党员积极参与徐汇校区迎新志愿服务，为新生提供各项服务和便利。学生党员统一身着学院迎新志愿者服装，胸前佩戴党徽与党员先锋队标志，以实际行动办实事，展现共产党员的精神面貌和服务意识。今年，还在新生志愿服务点设置了书写入党志愿书的纸笔领取点并选派优秀学生党支书现场驻点答疑，为新生提供便利的同时，帮助同学们进一步了解党的理论知识，对入党流程等相关问题答疑解惑，让新生感受党组织温暖，进一步彰显党员旗帜风采，激发红色动力。同时，抓住新生启蒙黄金期，以年级大会形式开展入党启蒙教育，从为什么入党、如何申请入党两方面具体展开介绍，帮助有志于加入共产党的同学清晰地了解入党流程，也鼓励同学们肩负起时代使命，积极向党组织靠拢。

有信仰，青春才不会无处安放。通过线上线下深度融合的双线联动模式，安泰学生党员为群众办实事，以实际行动展现青年学子的使命和担当，充分发挥一名党员就是一面旗帜的红色引领作用，帮助2021级新生们迎接崭新学习生活的同时，做好红色引领。生逢盛世，肩负重任，相信安泰学子中能够涌现出新一批怀抱伟大理想、有志于伟大事业的新生力量，不负使命，让青春与信仰同行。
