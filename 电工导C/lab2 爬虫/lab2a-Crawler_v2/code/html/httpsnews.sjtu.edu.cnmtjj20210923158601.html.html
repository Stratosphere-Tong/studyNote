上海这所高校大学生过农民丰收节[图]

2021年09月23日 责任编辑：刘婧怡 

新华网

新华网报道 酿葡萄酒、腌制泡菜......9月22日，在第四个中国农民丰收节来临之际，上海交通大学农业与生物学院选修新农科劳动教育课程的学生前往实验室，对前期自己栽种的劳动成果进行深加工，体验完整的葡萄酒酿造和蔬果腌制发酵流程。

　　上海交大农业与生物学院学生正在采摘葡萄。新华网史依灵 摄

　　在葡萄酒文化与酿造流程、食品发酵工艺两场简短的导论课后，同学们被划分为“酿酒”组和“腌泡菜”组，经专业老师的指导，分别体验不同的制作课程。

　　农业与生物学院大二学生李赛亚表示，身为农生学子，在埋首学习的同时，也应该走出书本，去感受一下真正的田间地头、加工生产的生活。“劳动教育恰好给了我这样一个可以近距离接触的机会。我想，这将是我一生中的难忘经历和宝贵财富。”李赛亚说。

　　上海交大农业与生物学院学生展示团队种植的葡萄。新华网史依灵 摄

　　新农科劳动教育课程开始于春分，并于秋分日开展丰收节主题活动。同学们以班级为单位认领责任田，组成生产大队，在老师和专业工人的指导下，分栽培管理、收获储藏、加工展出三个阶段完成生产劳作。

　　“从3月份春分刚过开始，在专业教师的指导下我们学习到番茄、葡萄的种植和产品加工技术知识，并以班级为单位承包了‘责任田’，之后的几个月里，大家挖沟起垄、悉心耕种，并在秋天收获了西瓜和番茄。”农业与生物学院大二学生孙梦鸽说，这一学期的劳动教育中，我切实体会到了“一粥一饭当思来之不易”。

　　上海交大农业与生物学院学生正在制作泡菜。新华网史依灵 摄

　　从春分播种到秋后收获，再到品尝自己亲手制作的葡萄酒与泡菜，这场特别的劳动实践“思政课”持续了七个月有余，学生们不仅体会到了农业生产的不易，更加坚定了强农兴农的决心。农业与生物学院大二学生农润芝说：“纯理论的专业课并不能让我们切身实地体会到劳动产品是如何产出。这个课程让我们从头到尾一点点去感受劳动产品是如何产生的，感受劳动人民的艰辛，提醒我们珍惜劳动成果。”

　　上海交大农业与生物学院学生正在酿制葡萄酒。新华网史依灵 摄

　　新农科劳动教育课程是上海交通大学依托农业与生物学院的学科特色，试点打造的 “1+3”的劳动育人模式，即以一门精品劳动教育思政课为核心，以实践劳动、专业劳动、科研劳动三种劳动教育形式为载体，将劳动教育纳入课时，形成第一课堂与第二课堂相融合的劳动教育模式。上海交大农业与生物学院助理研究员傅佩宁表示，新农科劳动教育课程开设以来在农生学院十分受欢迎，后续计划把这一课程扩大，设置成全校学生的公开选修课。
