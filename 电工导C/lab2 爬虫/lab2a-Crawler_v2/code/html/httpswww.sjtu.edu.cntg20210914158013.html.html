第二十届“文治杯”大学生写作大赛征稿启事 ( 2021年09月14日 )

传记书写人生，纪念人性，承载文化传统，传递人文精神。中国传记历史悠久，传记文类也在变革之中，同时代精神的变化紧密相连。中国现代传记自新文化运动以来，经过曲折发展，焕发新的生机。改革开放以后，进入创作和出版的繁荣期。 

当今中国正经历世界百年未有之大变局。回首峥嵘岁月，中华民族在风雨中走向复兴。人民是历史的创造者，时代精神和文化传统均汇聚于个体生命之中。无论是英雄模范、科学先驱、仁心医者、文化匠人等时代楷模，还是普通百姓、平凡人物，都有着充满活力、色彩丰富的人生，其生命体验和人格力量值得被记录传承。

上世纪初，交通大学唐文治校长重视国文教育，亲自组织一年一次的全校学生国文大赛（即作文比赛），成为学校教学一大特色。为传承唐文治校长国学教育理念，激发当代大学生写作积极性，提升大学生文化素质，上海交通大学国家大学生文化素质教育基地持续举办“文治杯”文学作品大奖赛，迄今已十九届。

第二十届“文治杯”大奖赛升级为全国大学生写作大赛。当代大学生成长于中国高速发展的新世纪，关注时代、感悟生命是青年人的责任，本次大赛的写作体裁为传记（生命写作），以“时代与个人”为主题，以真人实事为内容。现将有关事项通知如下：

一、 征稿范围及要求

1.征稿对象：全国在读大学生（本硕博均可，含留学生）。

2.征稿体裁：各类文字传记（生命写作），包括他传、自传、回忆录、游记、日记、书信等，题目自拟，字数在20000字以内，每人限投1篇。

3.截稿日期：2021年12月31日。

4.投稿方式：大赛仅接收电子文稿，请将作品以电子文档形式发送至邮箱:sjtuwenzhibei@126.com，邮件主题为文治杯+传记作品名+姓名+手机号+学校（单位）。应征作品请以附件形式发送。

5.参赛作品必须为本人原创，符合传记文类的要求，且未在各类公开出版物或网络发表，不得抄袭杜撰，严禁他人代写。一经发现违规行为，即取消参赛资格。

二、 奖项设置

本次大赛设一等奖1名、二等奖2名、三等奖5名、优秀奖10名。优秀作品将推荐发表。（详细信息）

三、大赛组织

主办单位： 上海交通大学国家大学生文化素质教育基地

上海交通大学传记中心
