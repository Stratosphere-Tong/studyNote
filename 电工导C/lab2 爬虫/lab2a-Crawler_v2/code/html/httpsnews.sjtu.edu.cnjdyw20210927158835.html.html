交大要闻

上海交通大学第三十三次学生代表大会召开

2021年09月26日 责任编辑：杨吴悦 

2021年9月25日，上海交通大学第三十三次学生代表大会在闵行校区菁菁堂召开，来自全校各院系的443名学生代表齐聚一堂。校党委副书记、副校长王伟明，学生工作指导委员会秘书长侯士兵，教务处处长杨颉，研究生院培养办主任孔令体，后勤保障中心党委副书记李梅，校团委书记钱文韬及教师代表出席大会。上海市学生联合会主席李腾、清华大学学生会执行主席刘至真等兄弟高校学生会同仁应邀参与。

上午，第一次全体会议在庄严的国歌声中拉开序幕。

李腾代表上海市学联致辞。他表示，上海交通大学学生联合会在学校党委的领导下，在校团委和市学联的指导下，在助力上海城市发展、领跑学联学生会组织改革、推动学校“双一流”建设中贡献了青春力量。他提出，希望上海交通大学学生联合会继续深入推进改革，发挥好党领导下的主要学生组织的桥梁纽带作用，汇聚青春智慧，点燃青春活力。他希望交大学子用火红的青春，为国家、为城市的发展做出更大的贡献，在实现中华民族伟大复兴的伟大实践中成就青春梦想、书写人生华章。

刘至真代表兄弟高校学生会致辞。他表示交大学联在引领青年思想、服务学生成长、繁荣校园文化、维护学生权益等方面，积极开展了一系列富有主题教育意义的校园活动，为学校事业发展和育人工作做出了重要贡献。两校的学生会始终保持着密切的联系，进行了多层次、有成效的合作交流。希望交大学联继承光荣传统，牢记初心使命，担当时代责任，勇于砥砺奋斗，站在新的历史方位再出发，充分展现交大青年的昂扬姿态，努力成为新时代的奋进者、开拓者和奉献者。

王伟明代表学校党委致辞。他向大会的顺利召开表示热烈的祝贺，向市学联、兄弟高校学生会及大会筹备工作组表示衷心的感谢，向与会的各位代表和全校青年学生致以诚挚的问候和美好的祝愿。王伟明高度肯定了第三十二届学生联合会开展的卓有成效的工作，同时对学联学生会提出期望。他表示，学联学生会改革已推向纵深，学联学生会要准确把握回归初心使命的内涵，明确职能定位，反映同学的根本诉求；面对时代要求和历史责任，学联学生会应锐意进取，团结更多青年学生，在学业、工作和品德方面成为广大同学的表率和标杆。他强调，一代青年有一代青年的使命，全体同学应坚定理想信念毫不动摇，以矢志艰苦奋斗践行时代使命，用实际行动践行“选择交大，就选择了责任”。

第三十二届学生联合会执行主席卫宇杰向大会作题为《开拓进取 勇担使命 在迈向世界一流大学前列的征程中书写青春篇章》的工作报告。报告从“紧扣时代主题，构建引领青年的多样态载体”，“探索改革实措，推动交大学联的内涵式发展”，“践行初心宗旨，加强服务品牌的体系化建设”三个方面系统地回顾了第三十二次学生代表大会以来学联的工作情况。报告中提出学联深化改革已经进入攻坚阶段，学联人必须以更大的勇气和智慧，推动学联组织机制改革和服务体系优化，切实保持和增强学联的政治性、先进性和群众性，不断提升学联工作的同学满意度、大局贡献度和社会认可度，开拓进取、勇担使命，团结带领广大交大同学为上海交通大学迈向世界一流大学前列、为实现中华民族伟大复兴的中国梦而共同奋斗。

大会听取了《上海交通大学第三十三次学生代表大会提案工作报告》《上海交通大学第三十三次学生代表大会常任代表候选人名单》，听取并审议了《上海交通大学学生联合会章程（修正案）（草案）》，通过了《上海交通大学第三十三次学生代表大会常任代表选举办法（草案）》《上海交通大学第三十三次学生代表大会常任代表选举总监票人、监票人名单（草案）》。

各分会场代表团分组讨论《上海交通大学第三十二届学生联合会工作报告》《上海交通大学学生联合会章程（修正案）（草案）》。随后，全体学习习近平总书记在庆祝中国共产党成立100周年大会上的重要讲话精神，并选举上海交通大学第三十三次学生代表大会常任代表。

下午，第二次全体会议正式召开。会议通过《上海交通大学第三十三次学生代表大会关于<上海交通大学第三十二届学生联合会工作报告>的决议（草案）》《上海交通大学第三十三次学生代表大会关于<上海交通大学学生联合会章程（修正案）>的决议（草案）》，通报上海交通大学第三十三次学生代表大会常任代表选举结果。丁雪等35名同学当选第三十三次学生代表大会常任代表。

大会进行上海交通大学第三十三届学生联合会主席团的选举。大会通过《上海交通大学第三十三届学生联合会主席团选举办法（草案）》《上海交通大学第三十三次学生代表大会总监票人、监票人名单（草案）》，听取《上海交通大学第三十三届学生联合会主席团正式候选人名单》。七位候选人按照抽签顺序依次发表演讲，阐述了自己的服务理念与工作计划，现场代表就各位候选人的展示积极提问，全面了解各位候选人。

最终，张配荧、杨行智、符瑶、木拜西尔·麦麦提依明、余瑜（按演讲抽签顺序排列）五人当选上海交通大学第三十三届学生联合会主席团成员，并与第三十二届学联主席团进行了学联会旗交接。

上海交通大学第三十三次学生代表大会闭幕。

同学们表示，“请党放心，强国有我！”，必将不辱时代使命，不负人民期望，坚定理想信念，培育高尚品格，练就过硬本领，勇于创新创造，同亿万人民一道，在奋斗中谱写新时代的青春之歌！
