交大要闻

对标中央巡视要求，推动校内巡察工作高质量发展

上海交通大学十一届党委第四轮巡察工作动员部署会召开

2021年09月28日 责任编辑：李伟 

根据学校党委统一部署，9月27日，上海交通大学十一届党委第四轮巡察工作动员部署会在闵行校区召开。校党委副书记、纪委书记、巡察工作领导小组副组长周承出席会议并作动员部署讲话。党委常委、组织部部长仰颐，巡察工作联席会成员单位负责人，巡察组和巡察办全体人员参加会议。会议由巡察办主任吴淑琴主持。

会议深入学习领会习近平总书记听取十九届中央第七轮巡视情况汇报时的重要讲话精神，贯彻中央巡视工作新精神新部署新要求，深化巡视巡察上下联动，精准落实政治巡察要求，有效推进校内巡察工作，对本轮巡察进行动员部署。

周承作动员部署讲话。他指出，开展校内巡察工作，要时时对标中央巡视，要把中央巡视反馈指出的问题作为内部巡察监督的重中之重，从政治高度把握新时代巡察工作的定位内涵、发展方向，发挥巡察政治监督作用。要聚焦习近平总书记和党中央关于加强和改进党对高校的全面领导、坚持社会主义办学方向、落实立德树人根本任务和深化从严管党治校等重要部署要求，从政治上查找和推动解决影响单位发展的深层次问题，督促被巡察单位认真履行职能职责，完整准确全面贯彻新发展理念，确保党的领导、“两个维护”真正落实落地。

周承强调，本轮巡察在做好“四个落实”监督的基础上，要把握不同单位职能责任，找准被巡察单位存在的突出问题，增强巡察的科学性精准性实效性，聚焦“六个着力”发现问题。一是着力检查贯彻习近平新时代中国特色社会主义思想，落实党的教育方针和党中央重大决策部署、学校党委工作要求情况。二是着力检查履行核心职能情况，紧盯深入谋划服务国家重大战略，提升学校对国家科技进步和世界影响力贡献度等问题。三是着力检查防范廉洁风险和规范权力运行情况，关注科研经费管理、基建招标采购等重点领域存在的廉洁风险。四是着力检查作风建设情况。五是着力检查“一把手”和领导班子建设，深化改革和防范化解重大风险情况。六是着力检查党组织建设情况。

周承要求，巡察组要持续学习贯彻习近平总书记关于巡视巡察工作的重要论述，不断提高政治判断力、政治领悟力、政治执行力。要提高思想认识，精准落实政治巡察要求。要增强工作合力，巡察工作联席会成员单位、巡察组和被巡察单位要密切配合。要严格纪律要求，坚持实事求是、依规依纪依法，确保如期圆满完成巡察任务。

会上，仰颐宣读了《关于中共上海交通大学第十一届委员会第四轮巡察组授权任职及任务分工的决定》。

经学校党委研究决定，十一届党委第四轮巡察单位为：科学技术发展研究院、基建处。
