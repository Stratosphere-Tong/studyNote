交大金课

首批国家级线上一流课程

江志斌：生产计划调度与控制

— 交大金课
2021年09月28日 责任编辑：余建波 张文清 孙佳 
2012年，全球慕课快速发展，上海交通大学于2013年在国内率先开展慕课建设及推广，截止目前，共上线在线课程（慕课）165门。教育部于2017年教育部开始认定国家级精品在线开放课程（即线上金课），2019年开始认定国家级一流课程（五类金课）。截至目前，我校共有55门课程被认定为国家级一流课程（金课）。

【金课名称】生产计划调度与控制

【任课教师】江志斌

【名师名片】

江志斌，上海交通大学安泰经管学院特聘教授、董浩云航运与物流研究院院长，教育部长江学者特聘教授，国际工业与系统工程师学会副主席、Fellow，教育部工业工程教学指导委员会副主任委员，中国机械工程学会工业工程专业委员会副理事长，上海机械工程学会工业工程专业委员会主任委员，国际生产研究等5个国际期刊副主编或编委；原为上海交大工业工程与管理系主任，曾任学校教务处处长、慕课推进办主任、慕课研究院院长。长期从事工业工程专业课程教学研究与课程建设，积极推进工业工程专业改革，先后获得国家级教学团队、5门次国家精品课程、2项国家级教学成果奖、3项上海市教学成果一等奖，获得宝钢全国优秀教师奖和宝钢优秀教师特等奖，为国务院特殊津贴专家。

【课程介绍】

本课程是工业工程专业核心课程，也可用于物流工程、管理科学与工程以及工商管理等本科人才培养。课程的主要内容包括生产系统概念、需求预测、库存管理、生产计划、作业调度以及精益生产等，目的是加深学生对生产管理的理解，掌握生产管理中的核心理论和方法。为了加强学生解决实际问题的能力，配备了拓展阅读和案例供学生讨论，增加了课程设计内容。

本课程原名为《生产计划与控制》，2007年被评为教育部精品课程，英文版于2013年被评为教育部来华留学生精品课程，2016年又被评为教育部精品资源共享课程。为了在更大范围内实现课程资源共享，结合在线课程教学和混合式教学的需要，经过课程再设计和内容优化，建成在线课程，并更名为《生产计划调度与控制》，于2016年秋季正式在“好大学在线”平台上线，并持续开展，2020年入选国家首批“双万计划”国家级一流本科课程。首先在上海交通大学推进线上与线下同步的教学模式，我校工业工程与管理系的本科生中共200多人参与，受到了众多学生的欢迎和好评。此外，还吸引了来自一百多所高校近四千人在线注册和学习该课程。从2019-2020学年春节学期起，该课程作为安泰经管学院本科生课程《运营管理》混合式教学线上课程内容，进行了线上线下混合教学，取得了较好的效果。
