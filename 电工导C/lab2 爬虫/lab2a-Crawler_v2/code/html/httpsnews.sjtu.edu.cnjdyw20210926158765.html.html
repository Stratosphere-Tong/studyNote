交大要闻

“品物游心：中国文人的生活与艺术”展在上海交大开幕

上海交通大学档案文博管理中心档案与文物保护技术室揭牌

2021年09月26日 责任编辑：徐骞 

俯仰天地间，乘物以游心。9月24日，由上海交通大学和上海博物馆主办、上海交通大学博物馆承办的“品物游心——中国文人的生活与艺术”展在上海交通大学闵行校区文博楼拉开帷幕。上海博物馆馆长杨志刚、上海大学党委副书记段勇等校外来宾，上海交通大学校长林忠钦、原党委书记王宗光、原校长谢绳武、原党委书记马德秀、党委常委、副校长张安胜等学校领导，以及上海博物馆展览团队、相关博物馆及兄弟院校领导，上海交通大学博物馆馆长陈克伦、钱学森图书馆馆长钱永刚等学校有关院系部处和直属单位的领导及师生代表100余人出席了开幕式。

展览选取“幽斋栖志”“艺文寄兴”“格物得趣”“燕闲怡情”四个维度，遴选了来自上海博物馆馆藏的125件/套文物展品，通过读物与情景再现相结合的方式，展现了中国文人的风雅生活与精神旨趣，让观众体悟古贤之性情、追仰诸君之品格，进而关照自身之心灵。

杨志刚在开幕致辞中指出，上海交通大学与上海博物馆长期保持着深厚的友谊，从调研互访、学术交流到签订战略合作协议，此次展览是双方耕耘多年的一次丰收。作为国内顶尖的高等学府和博物馆，双方肩负着以文化人、以文育人、以文培元的重要使命。期待双方在未来开展更多层次的合作与交流，深入挖掘中华优秀传统文化中的文化价值和审美价值，充分发挥其文化滋养作用；携手前行，厚植城市文化底蕴，阐释好中国精神、中国价值、中国力量。

为纪念上海交通大学与上海博物馆的首个合作大展的举办，向为此次展览辛勤付出的上海博物馆同仁表示感谢，张安胜代表学校向杨志刚赠送了“品物游心”展览纪念杯。

近年来，随着博物馆事业的快速发展，对文物藏品的保存、保护也成为学校关注的重点。为进一步加强学校档案与文物的保护修复工作，学校在2019年的“双一流”文博能力提升专项中，正式立项建设上海交大第一个专业的档案文物保护机构——“档案文博管理中心档案与文物保护技术室”。经过一年多的认真筹建，总用房面积近260平米的保护技术室也于当日落成，并由林忠钦、王宗光、谢绳武、马德秀共同揭牌。据悉，目前已完成的一期建设，配备了文物拍摄、仿真复制、检测修复、装裱加工等各类专业设备，未来将重点开展纸质文物的保护与修复工作。保护技术室的二期建设也正在规划中。

林忠钦在讲话中指出，上海交通大学自建校以来，始终将文理兼修、科艺相济作为学生的培养目标，在培育学生科学素养的同时，为其注入人文精神的养分。近年来，学校更是将文博育人作为开展传统文化教育的长效载体。希望作为全校文博事业新格局的核心的文博楼，继续加强与国内一流博物馆、兄弟院校及社会各界的交流与合作，为公众提供多样化的公共文化服务，为城市文明注入深厚的人文内涵，为时代与未来发展赋能。

随着林忠钦、杨志刚、王宗光、谢绳武、马德秀、段勇、张安胜、陈克伦、钱永刚共同推动推杆，“品物游心——中国文人的生活与艺术展”的长卷缓缓打开，标志着展览正式启幕。

与会领导与嘉宾一同参观了“品物游心——中国文人的生活与艺术展”，由本次展览策展人、上海博物馆工艺部主任施远做专题导览。

作为传承与弘扬优秀传统文化的阵地和载体，上海交通大学博物馆将继续深耕校内外资源，打造高品质、多元化的文博品牌，切实推进“以文化人，以史育人”之目标，进一步凝聚和培育新时代的交大精神，以传承文明、探求真理为使命，以振兴中华、造福人类为己任，不断开启新的征程，创造新的辉煌。
