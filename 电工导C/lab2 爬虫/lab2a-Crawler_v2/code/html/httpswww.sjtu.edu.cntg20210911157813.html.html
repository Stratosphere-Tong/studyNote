上海交通大学关于2021年中秋、国庆放假及调课安排的通知 ( 2021年09月11日 )

各院（系）、部、处、直属单位：

根据国务院办公厅关于2021年部分节假日安排的相关通知，结合我校校历安排和实际情况，现将我校2021年中秋节、国庆节放假调休及调课安排通知如下。

一、中秋节：9月19日至21日放假调休，共3天。9月18日（星期六）上班，教学安排按照2021－2022学年秋季学期第二教学周的周一课表执行；

二、国庆节：10月1日至7日放假调休，共7天。9月26日（星期日）、10月9日（星期六）上班。其中9月26日的教学安排按照2021－2022学年秋季学期第四教学周的周二课表执行，10月9日的教学安排按照2021－2022学年秋季学期第四教学周的周四课表执行。

辅修教学按上述安排执行。非全日制专业学位研究生放假期间的课程安排由研究生院另行通知。医学院、上海高级金融学院、交大密西根学院、继续教育学院、海外教育学院放假期间的课程安排，分别由以上各单位另行通知。

各单位要认真做好相关工作的安排，教务处、研究生院、各院（系）将对放假调课期间的教学进行巡查。放假期间，各单位要加强值班，做好常态化疫情防控和安全管理各项工作；后勤部门要做好师生员工的后勤服务工作；校区间交通班车按双休日班车时间运行。

特此通知。

上海交通大学

2021年9月9日
