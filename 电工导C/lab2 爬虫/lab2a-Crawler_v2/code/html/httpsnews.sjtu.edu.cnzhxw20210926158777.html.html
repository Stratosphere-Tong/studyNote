马克思主义学院研究生思政课教学团队以“共享、共情”重构教学共同体打造有深度、有温度的思政课

2021年09月26日 责任编辑：任祝景 

“让思政课成为交大学生真心喜爱、终生受益的课程，让交大的思政课改革成为全国贯彻大思政理念的试验田。”为进一步落实思想政治理论课课程体系改革建设，马克思主义学院研究生思政课教学团队按照“共享、共情”团队原则，以期充分发挥教师的积极性、主动性、创造性，探求研究生思政课教学中的学术性与思想性、科学性与价值性、时代性与历史性，通过案例教学与分组讨论的教学模式创新，打造出有深度有温度，受学生喜爱，具有获得感的研究生思政课，研究生思政课教学团队由吴红、李主斌、龙钰、魏华和史斌五位教师组成。

 “共享”。教学团队实现备课资源、教学方法、教学进度的“共享”。暑期，教学团队先后组织召开5次集体备课会，讨论课程整体性安排，对教学规范和教学内容进行优化调整，针对团队每位教师的PPT、案例、课程导入等方面进行了系统安排，确保团队成员掌握彼此教学进程。

 “共情”。教学团队认为传统的教学形式以老师为主体，老师讲什么，学生听什么，缺乏双向互动。根据本校研究生知识面宽、兴趣爱好丰富等特点，要想满足学生对思政课教学的要求。本教学团队设计出“有理、有情、有趣”的课程模块。“有理”即以“问题导入”，课程前通过“问卷星”“学习通”技术手段，设计课程调查问卷，以便掌握学生期待话题以及对于本学期课程的期望值和教学内容、教学形式。把培养学生的思维、方法、表述、理论与实践结合的能力贯穿于整个教学过程；“有情”即以学生为主体，学情分析为基础，通过典型案例达到与学生的“共情”，以进一步形成价值认同；“有趣”即以分组讨论形式，由学生自由组合成讨论组，相互介绍，为本讨论组取组名、选队长，以寓教于乐的方式培养学生思考、主动提出问题，增强老师与学生之间的互动，老师有针对性的进行引导，以此打造一套全方位、立体的课程教学模式。

2021年9月16日，“新时代中国特色社会主义理论与实践”2021-2022学年秋季学期第一次课程开课，次日，“中国马克思主义与当代”教学团队也开课。

通过本学期“研究生思政课程”团队教学的开展，打破了以前学生对思政课的刻板印象，学生反响热烈。教学团队明确了“构建适合青年学生全面发展、契合时代发展的思政课程体系”的宗旨，在今后教学实践中将不断创新教学模式，将理论内容通俗化，将单一的授课形式多样化、乐趣化，让思政课教育真正地入脑入心。
