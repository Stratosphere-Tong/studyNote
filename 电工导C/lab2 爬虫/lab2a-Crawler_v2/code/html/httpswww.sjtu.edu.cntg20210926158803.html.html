通知公告

关于2021年度上海交通大学思政创新发展研究课题申报工作的通知 ( 2021年09月26日 )

各有关单位：

为全面贯彻落实习近平总书记关于教育的重要论述，不断加强高校辅导员队伍建设，进一步提升高校思想政治工作研究质量，推动高校加快构建思想政治工作体系，切实形成全员全过程全方位育人格局，经研究，决定发布2021年度上海交通大学思政创新发展研究课题，现将有关事项通知如下。

一、项目情况

（一）项目类型

本次《课题指南》（附件1）分为理论研究和实践研究两个类型。

1. 理论研究。理论研究包括大学生成长成才和网络育人两个方向。理论研究应根据指南选题，自选角度进行研究，可以围绕选题指南适当调整题目。

2. 实践研究。实践研究课题应围绕学生党建工作、思想政治教育、学生管理与服务、思政工作队伍建设和大学生心理健康等方面的重点和难点问题进行研究，申报人可从课题指南中自选角度，或立足工作自拟题目。

（二）项目要求

理论研究课题完成时间根据申报人评估后协商确定，原则上要求2023年12月31日前完成，成果形式为专著书稿、高水平论文等。实践研究课题要求2022年12月31日前全部完成，成果形式为高水平论文、省部级以上课题、决策咨询报告、制度文件、工作指南、研究报告、网络评论文章、网络文化产品等。

（三）资助额度

资助经费根据个人申报和评审结果共同确定，原则上每项理论研究课题总计给予不超过10万元经费支持，每项实践研究课题给予不超过3万元经费支持。根据课题进展情况，分次资助。

二、申报要求

1. 本次学校思政创新发展研究课题申报，采用公开发布、个人申报、部门审核、专家评审的方式进行。

2. 申报人须具有较高的政治素质、深厚的学术造诣和丰富的科研经验。原则上，理论研究课题申报者应有副高级（含）以上专业技术职务或副处级（含）以上行政管理职务，能够承担实质性研究工作并担负科研组织职责。实践研究课题申报者应有中级（含）以上专业技术职务或副科级（含）以上职务。挂名或不承担实质性研究工作的人员不得作为课题组成员参与课题申报。

3. 申报人只能申报一项课题，且不能作为课题组成员参与本次其他课题的申报，课题组成员最多可参与两个课题。如申报人已有在研的思政创新发展课题，则本次不可申报。

4. 本次申报需在2021年10月26日（星期二）前完成系统填报和纸质提交，逾期不再受理。

（1）电子版。凭本人Jaccount登录大发院课题管理平台，完成填写和上传电子《课题申报书》（附件2），系统开放时间为2021年10月8日至2021年10月26日下午23时59分。

（2）纸质版。从课题管理平台下载填好的《课题申报书》，A4纸双面打印1份，经分管领导签字、所在单位科研管理部门审核盖章（签字）后报送新行政楼B307室。

三、后续安排

（一）评审立项

申报截止后将进行校内外专家匿名评审工作，通过立项评审的课题组将收到立项通知书。

（二）过程管理

为确保课题质量，大学生发展研究院将实施周例会和月报告制度。自课题立项第二个月起，由课题负责人定期组织开展学术研讨、课题交流等活动，并在系统提交月报。大学生发展研究院将于项目中期组织中期检查。

（三）结题要求

1. 对照立项申报书约定的目标任务进行验收考核，成果第一署名应为本课题负责人；

2. 论文为相关学术期刊上公开发表的有关本课题的相关研究成果，须含期刊的封面页、目录页、正文、及封底页等复印件（如论文已交杂志社待发，须提供相应用稿证明）；

3. 专著为与课题相关的正式出版物，须含封面页、编委会页、目录页和封底页等复印件（如著作权已交出版社待出，须提供相应出版合同复印件）；

4. 决策咨询报告或工作建议应与课题相关，须列明报告及建议的名称、报送单位、是否获领导正面批示或建议采纳，并提供相关证明；

5. 理论研究课题若未能完成立项申报书约定的预期成果，则至少提交一份不少于3万字的课题研究报告。实践研究课题若未能完成立项申报书约定的预期成果，则至少要提交一份不少于8000字的课题研究报告；

6.所有成果应注明“上海交通大学大学生发展研究院”或“教育部高校思想政治工作创新发展中心（上海交通大学）”相关字样（含课题名称和立项编号）。

（四）提前结题

已完成研究内容且符合结题要求的课题，有意愿申请提前结题的，须由课题负责人书面提交《提前结题申请表》，经所在单位（部门）审核后，报大学生发展研究院办公室申请，获批后按要求提交结题材料。

（五）延期申请

课题负责人应按照《课题申报书》约定的时间完成课题研究任务，如需延长课题研究时间，须由课题负责人书面提交《思政创新研究课题延期备案表》，经所在单位（部门）审核后，报大学生发展研究院办公室办理。延期结题申请最多一次不得超过半年，同一课题延期结题申请次数最多不得超过2次，到期仍未完成者将予以撤项。
