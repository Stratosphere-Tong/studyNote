上海交通大学2021年新任班主任代表座谈会召开

2021年09月26日 责任编辑：汤翔鹰 

9月22日，上海交通大学2021年新任班主任代表座谈会在行政楼会议室召开。校长、中国工程院院士林忠钦出席会议讲话，党政办、党委教工部、研究生院、教务处、学指委等部门同志和来自不同院系的14名班主任代表参会。校党委副书记、副校长王伟明主持会议。

林忠钦在讲话中指出，班主任对于学生的影响至关重要，班主任要成为学生最信赖的群体。他对各位新任班主任寄予厚望，提出三点要求：一是帮助学生树立正确的人生观和远大志向；二是激发学生对与所学专业的自信和热爱；三是指导学生应对学习和生活中所遇到的困难，以知心朋友的身份关注年轻人的心理问题。他强调，班主任要意识到育人工作的神圣性，重视对学生学术志趣的培养，为国家和社会重点发展领域培养一流的人才。

学指委秘书长、学生处处长侯士兵介绍了班主任队伍的总体情况。

材料科学与工程学院院长、讲席教授孙宝德作为2020级班主任代表发言。他结合自己的班主任工作经历向各位新任班主任分享工作经验，从提高学生自我管理能力、引导学生树立正确的奋斗心态和激发学生对专业的热爱三个方面，鼓励各位新任班主任发挥自身特长和优势，做好学生成长成才的引路人。

船舶海洋与建筑工程学院教授马宁，机械与动力工程学院教授饶宇，电子信息与电气工程学院教授陶梅霞，物理与天文学院教授朱卡的，外国语学院教授尚必武作为2021年新任班主任发言，分享了自己对班主任工作的理解和思考。

现场老师们积极交流了自己对于班主任工作的体悟与建议。

据悉，自2018年推进高层次人才担任班主任以来，累计119名高层次人才主动申请担任班主任，形成了全员育人的良好氛围。下一步，学指委将持续开展班主任工作沙龙，着力提升班主任班级管理能力，配齐班主任日常工作“工具箱”。
