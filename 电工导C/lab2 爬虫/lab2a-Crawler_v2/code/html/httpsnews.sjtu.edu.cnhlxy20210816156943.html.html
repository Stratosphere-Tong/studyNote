错过了流星雨？别担心，交大璀璨星河助你许愿！

2021年08月15日 责任编辑：曾玉竹 

凌晨，盛大的流星雨在北半球出现

你是否许下了夏末初秋的心愿？

也许沥沥雨声、绵绵云层遮挡了星光

但没关系，美景就在你身边

交大校园的美妙星夜

时常上演肉眼可见证的奇迹

现在，就让我们一起见证

梦想中的交大星河！

 

流星·荧荧心愿

一鼓作气，划破地球的外衣

燃烧光与热，融为灰烬

承载着无数心愿，书写了奇迹

在交大

努力就是奇迹的另一个名字

星轨·时间足迹

沉默地行走着

一路与交大相伴

时间，画出追逐的轨迹

奋斗，向着未至之境

银河·浩瀚天幕

湖泊、夜幕、星河、宇宙

香甜睡梦里

我曾畅游星空

浩瀚苍穹中

我拉开梦想的帷幕

仰望，收藏一串星光

前行，为着实现愿望

在交大，抬头就有一片星空

那是师生智慧的光芒

挥洒的青春
