交大要闻

中国工程院院士黄震做客焦点讲坛畅谈碳中和愿景下的能源变革与大学责任

2021年09月24日 责任编辑：孙佳 

9月22日下午，能源与动力工程专家、中国工程院院士、智慧能源创新学院院长、原副校长黄震做客上海交大第129期焦点讲坛，以“碳中和愿景下的能源变革与大学的责任”为题作专题报告。校党委常务副书记顾锋，校党委常委、党委宣传部部长胡昊，校院两级党组织理论学习中心组成员、师生代表、部分机关干部、思政教师等参加学习。报告会由顾锋主持。

黄震从全球气候变暖与温室气体排放控制、碳中和愿景下的能源变革、大学的责任三方面进行了专题报告。黄震首先强调了实现碳达峰、碳中和是一场广泛而深刻的经济社会系统性变革，对我国建设人与自然和谐共生的社会主义现代化强国具有重要战略意义。他回顾了能源技术进步对人类社会发展和工业革命的重要性，并针对2030年前二氧化碳排放达到峰值、2060年前实现碳中和的目标，阐述了碳中和愿景下能源变革与发展大趋势，并围绕大学责任进行了分析。他表示，学校要在变革中积极作为，一方面充分发挥跨学科优势，持续推进能源科技创新，主动服务国家战略和地方需要；另一方面加快培养低碳行业专业人才，为我国低碳转型发展提供人才保障、专业支撑和技术储备，为人与自然和谐共生的社会主义现代化强国建设作出积极的贡献。

报告结束后，黄震与现场师生进行了互动交流。

实现“碳达峰”“碳中和”是以习近平同志为核心的党中央站在中华民族永续发展和构建人类命运共同体的高度，经过深思熟虑作出的重大战略决策，不仅彰显了我国对建设全球生态文明、构建人类命运共同体的责任担当，也是推动我国能源结构、产业结构、经济结构转型升级的自身发展需要。交大将勇担服务国家时代使命，开启迈向世界一流大学前列的新征程，为夺取全面建设社会主义现代化国家新胜利作出交大人新的更大的贡献。
